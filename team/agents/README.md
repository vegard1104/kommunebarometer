## Anbefalt oppstart

Ikke aktiver alle 9 på dag 1. Forslag til rekkefølge:

**Fase 1 (minste levedyktige team):**
1. Prosjektleder
2. Tech Lead
3. Fullstack-utvikler
4. UX/UI-designer
5. Release Manager

**Fase 2 (når Fullstack blir flaskehals):**
6. Frontend-utvikler
7. Backend-utvikler

**Fase 3 (når vi har noe i produksjon som må driftes):**
8. QA / Testing
9. DevOps / Platform

Prosjektleder kan selv si fra når det er på tide å utvide teamet.
